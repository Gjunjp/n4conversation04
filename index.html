文法ぶんぽう 1

〜てもいいですか

( 請問可以...嗎？ / 徵求許可 )

相手に許可（きょか）を求める時に使います。

動詞て形 ＋ もいいですか
(向對方尋求許可時使用。是日常辦公室與生活中非常實用的禮貌句型。)

許可をもらいましょう！ (來徵求許可吧！)

「〜てもいいですか」に変えてください。

1. ここに座すわります。(坐這裡)
➔ ここに ＿＿＿＿＿＿＿＿＿＿ 。(請問可以坐這裡嗎？)

2. 窓まどを開あけます。(開窗戶)
➔ 窓まどを ＿＿＿＿＿＿＿＿＿＿ 。(請問可以開窗戶嗎？)

許可をもらいましょう！ 解答

「〜てもいいですか」に変えてください。

1. ここに座すわります。(坐這裡)
➔ ここに 座すわってもいいですか 。

2. 窓まどを開あけます。(開窗戶)
➔ 窓まどを 開あけてもいいですか 。

文法ぶんぽう 2

〜ようと思おもっています

( 我正打算... / 我想... )

自分の計画（けいかく）や意志（いし）を伝える時。

（N4文法）同僚に自分の予定を話す時によく使います。
(用來表達自己的計畫或打算。在職場中告訴同事自己準備要做什麼時非常實用。)

動詞 意向形（よう/おう） ＋ と思っています

例れい：今日のお昼ひるは、ラーメンを食たべようと思おもっています。

計画を伝えよう！ (來表達計畫吧！)

請將以下動作改為「我正打算要... / 我想...」的句型。

  • 週末はゆっくり休みます。 (週末好好休息)
    ➔ 週末はゆっくり                               
  • 会議の資料を作ります。 (製作會議資料)
    ➔ 会議の資料を

会話かいわ 1：ランチに誘さそう (邀請吃午餐)

林： 田中たなかさん、お昼ひる、あのラーメン屋やに行いこうと思おもっているんですが、一緒いっしょにどうですか。
(田中先生，午餐我正打算去那家拉麵店，你要不要一起去？)
田中たなか： いいですね。でも、あそこのラーメンは辛からすぎると聞ききましたよ。
(好啊。但是我聽說那裡的拉麵太辣了喔。)
林： 大丈夫だいじょうぶです！辛からくないラーメンもあるそうですから。
(沒問題！聽說也有不辣的拉麵喔。)
